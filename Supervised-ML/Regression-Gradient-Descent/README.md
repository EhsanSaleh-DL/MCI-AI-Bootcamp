# Movie Gross Prediction using Gradient Descent Regression

## Overview

This project aims to predict the **Gross** revenue of movies using a Gradient Descent-based Linear Regression model. The dataset was preprocessed, divided into training and testing sets, and the model was evaluated using the Mean Squared Error (MSE) metric.

## Dataset

- Movie Dataset (movies.xlsx)

## Objectives

- Load the dataset
- Perform data preprocessing
- Handle missing values and categorical features
- Split the dataset into training (80%) and testing (20%)
- Implement Linear Regression using Gradient Descent
- Predict the **Gross** revenue
- Evaluate the model using Mean Squared Error (MSE)
- Investigate whether the movie title feature should be removed

## Algorithm

- Linear Regression
- Gradient Descent Optimization

## Evaluation Metric

- Mean Squared Error (MSE)

## Libraries

- NumPy
- Pandas
- Matplotlib
- Scikit-learn
