# Face Recognition using SVM, Random Forest, and AdaBoost

## Overview

This project performs face recognition using the Olivetti Faces dataset available in Scikit-learn. Three supervised learning algorithms were implemented and compared.

## Dataset

- Olivetti Faces Dataset
- sklearn.datasets.fetch_olivetti_faces()

## Objectives

- Load the Olivetti Faces dataset
- Split the data into training (80%) and testing (20%)
- Visualize one image from each person
- Train an SVM classifier
- Train Random Forest and AdaBoost classifiers
- Compare the performance of all models

## Algorithms

- Support Vector Machine (SVM)
- Random Forest
- AdaBoost

## Evaluation Metric

- Accuracy

## Libraries

- NumPy
- Matplotlib
- Scikit-learn
