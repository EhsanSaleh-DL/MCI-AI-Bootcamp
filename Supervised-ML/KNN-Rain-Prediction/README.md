# Rain Prediction using K-Nearest Neighbors (KNN)

## Overview

This project aims to predict whether it will rain tomorrow using historical weather data. The K-Nearest Neighbors (KNN) algorithm was implemented and evaluated using the F1-score metric to determine the optimal value of **K**.

## Dataset

- Weather dataset (BOM.csv)

## Objectives

- Load and preprocess the dataset
- Handle missing values and categorical features
- Split the dataset into training (80%) and testing (20%) sets
- Train KNN classifiers with different values of K
- Evaluate each model using the F1-score
- Select the optimal K
- Compare KNN with Decision Tree for real-time prediction

## Algorithms

- K-Nearest Neighbors (KNN)

## Evaluation Metric

- Precision
- Recall
- F1-score

## Libraries

- NumPy
- Pandas
- Matplotlib
- Scikit-learn
